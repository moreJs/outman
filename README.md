outman
======

this is also a amazing store.

这个系统是为了解决汽车生产企业冲压过程中的停机问题而设计的，严格来讲是一个专家系统，但是和传统的专家系统又有区别。

本系统采用 node + angularjs + MongoDB 的技术架构，构成了一个典型的单页应用。

这个版本目前是1.0 ， 只实现了数据字典的管理 ，但是大的框架已经搭建完毕了。目前在不断的进行中。。。
